\hypertarget{md__r_e_a_d_m_e_autotoc_md0}{}\doxysection{Link\+Auto}\label{md__r_e_a_d_m_e_autotoc_md0}
Link\+Auto es una red social para entusiastas de automóviles que permite a los usuarios conectarse, compartir contenido relacionado con vehículos y construir una comunidad en torno a su pasión por los coches.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysubsection{Requisitos previos}\label{md__r_e_a_d_m_e_autotoc_md1}

\begin{DoxyItemize}
\item Java 11 o superior
\item Maven 3.\+9 o superior
\item My\+SQL 8.\+0 o superior (o base de datos alternativa compatible)
\item Navegador web moderno (Chrome, Firefox, Safari, Edge)
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysubsection{Estructura del proyecto}\label{md__r_e_a_d_m_e_autotoc_md2}
El proyecto está dividido en dos módulos principales\+:
\begin{DoxyItemize}
\item {\ttfamily rest-\/api}\+: Backend basado en Spring Boot que proporciona los servicios REST
\item {\ttfamily client}\+: Frontend que consume los servicios del backend
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysubsection{Instalación y configuración}\label{md__r_e_a_d_m_e_autotoc_md3}
\hypertarget{md__r_e_a_d_m_e_autotoc_md4}{}\doxysubsubsection{1. Configuración de la base de datos}\label{md__r_e_a_d_m_e_autotoc_md4}
Cree una base de datos My\+SQL utilizando el script SQL proporcionado\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{rest-\/api/resources/dbsetup.sql}

\end{DoxyCode}


Este script creará\+:
\begin{DoxyItemize}
\item La base de datos de Link\+Auto
\item Un usuario {\ttfamily la} con contraseña {\ttfamily la} con los permisos necesarios
\end{DoxyItemize}

\begin{quote}
{\bfseries{Nota}}\+: Si desea utilizar credenciales diferentes, deberá modificar el archivo {\ttfamily rest-\/api/resources/application.\+properties} con los nuevos valores. \end{quote}
\hypertarget{md__r_e_a_d_m_e_autotoc_md5}{}\doxysubsubsection{2. Configuración alternativa de base de datos (opcional)}\label{md__r_e_a_d_m_e_autotoc_md5}

\begin{DoxyItemize}
\item {\bfseries{Error de conexión a la base de datos}}\+: Verifique las credenciales y la disponibilidad del servidor de base de datos.
\item {\bfseries{Error de puertos en uso}}\+: Asegúrese de que los puertos 8080 y 8081 no estén siendo utilizados por otras aplicaciones. Si es necesario, puede cambiar los puertos en los archivos de configuración.
\item {\bfseries{Problemas de compilación}}\+: Asegúrese de tener instalada la versión correcta de Java y Maven.
